## 🐛 常见问题排查

### 问题 1: 后端启动失败

**症状**: `mvn spring-boot:run` 报错

**排查步骤**:
1. 检查 MySQL 是否启动
   ```bash
   mysql -u root -pAnxiety200695 -e "SELECT 1"
   ```
2. 检查数据库 `life_hub` 是否存在
   ```sql
   SHOW DATABASES;
   ```
3. 检查端口 8080 是否被占用
   ```bash
   netstat -ano | findstr :8080  # Windows
   lsof -i :8080                  # Mac/Linux
   ```
4. 查看后端日志中的错误信息

### 问题 2: 前端显示"后端服务未连接"

**症状**: 首页底部显示红色圆点

**排查步骤**:
1. 确认后端已启动（访问 http://localhost:8080/api/health）
2. 检查浏览器控制台（F12）是否有 CORS 错误
3. 确认 `request.js` 中的 baseURL 正确
4. 检查防火墙是否阻止了 8080 端口

### 问题 3: 创建记录失败

**症状**: 点击"创建记录"后报错

**排查步骤**:
1. 打开浏览器控制台（F12）查看网络请求
2. 检查请求 URL 和参数是否正确
3. 查看后端日志中的错误
4. 确认数据库连接正常

### 问题 4: 分页不显示

**症状**: 创建了很多记录但没有分页按钮

**排查步骤**:
1. 检查后端返回的 `totalPages` 是否大于 1
2. 查看控制台是否有 JavaScript 错误
3. 确认后端分页逻辑是否正确

### 问题 5: 点赞后数量没有增加

**症状**: 点击点赞后没有反应

**排查步骤**:
1. 查看浏览器控制台的网络请求
2. 检查是否返回"已经点赞过"
3. 确认后端的点赞逻辑是否正确
4. 查看数据库 `character_interactions` 表
