# 快速启动指南

## 使用 Docker Compose 一键启动（推荐）

### 1. 确保已安装 Docker 和 Docker Compose

```bash
# 检查 Docker 版本
docker --version
docker compose version
```

### 2. 启动所有服务

```bash
# 在项目根目录执行
docker compose up -d

# 查看日志
docker compose logs -f
```

### 3. 访问应用

- 前端页面：http://localhost
- 后端 API：http://localhost:8080
- MySQL 数据库：localhost:3306

### 4. 停止服务

```bash
docker compose down

# 删除数据库数据
docker compose down -v
```

## 本地开发模式

### 后端启动

```bash
cd backend

# 确保 MySQL 已启动并创建数据库
mysql -u root -p < ../schema.sql

# 启动 Spring Boot
mvn spring-boot:run
```

### 前端启动

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问：http://localhost:5173

## 常见问题

### Q: 启动后前端显示"后端服务未连接"？

A: 等待 20-30 秒，后端需要时间启动。查看日志：`docker compose logs backend`

### Q: 数据库连接失败？

A: 确保 MySQL 容器已完全启动，查看日志：`docker compose logs mysql`

### Q: 端口冲突？

A: 修改 `docker-compose.yml` 中的端口映射，例如 `"8081:8080"`
