## 🎨 前端完成情况

所有页面已完成并可使用，包含完整的 CRUD 功能和后端 API 对接。

## 📄 页面功能详解

### 1. 首页 (Home.vue)

**路由**: `/`

**功能**:
- 展示项目介绍和三大功能模块
- 后端健康检查（实时显示后端连接状态）
- 导航到各功能模块

**特色**:
- 暖色调设计（terracotta 主题色）
- Hero Section 展示
- 功能卡片网格布局

---

### 2. 成长记录 (Growth.vue)

**路由**: `/growth`

**功能**:
- ✅ 查看成长记录列表（分页显示）
- ✅ 创建新记录（标题、内容、日期、学习时长、标签）
- ✅ 编辑已有记录
- ✅ 删除记录（带确认提示）
- ✅ 标签展示（逗号分隔，自动解析为标签）

**操作流程**:
1. 点击"新建记录"打开弹窗
2. 填写标题（必填）、内容、日期、学习时长、标签
3. 点击"创建记录"保存
4. 记录卡片显示在列表中
5. 点击"编辑"可修改，点击"删除"可删除

**API 调用**:
- `GET /api/growth?userId=1&pageNum=1&pageSize=10` - 获取列表
- `POST /api/growth` - 创建记录
- `PUT /api/growth/{id}` - 更新记录
- `DELETE /api/growth/{id}?userId=1` - 删除记录

**测试数据示例**:
```
标题: 学习 Spring Boot
内容: 今天学习了 MyBatis Plus 和 Spring Security
日期: 2026-07-01
学习时长: 120
标签: Java, Spring Boot, 后端开发
```

---

### 3. 角色卡片 (Characters.vue)

**路由**: `/characters`

**功能**:
- ✅ 浏览所有公开角色（卡片网格布局）
- ✅ 创建新角色（完整的角色属性）
- ✅ 查看角色详情（点击卡片，浏览量自动+1）
- ✅ 点赞角色（防重复点赞）
- ✅ 属性可视化（攻击/防御/速度进度条）
- ✅ 星级评分显示

**角色属性**:
- 基础信息：名称、原型人物、头像（Emoji）、称号
- 等级系统：等级、星级（1-5星）
- 三维属性：攻击力、防御力、速度
- 技能系统：技能名称、技能描述
- 故事背景：背景故事、性格标签、名言
- 统计数据：浏览量、点赞数

**操作流程**:
1. 点击"创建角色"打开弹窗
2. 填写角色信息（名称必填）
3. 选择星级、设置属性值
4. 填写技能和背景故事
5. 点击"创建角色"保存
6. 卡片显示在网格中
7. 点击卡片查看详情
8. 点击❤️按钮点赞

**API 调用**:
- `GET /api/characters?pageNum=1&pageSize=10` - 获取角色列表
- `GET /api/characters/{id}` - 获取角色详情
- `POST /api/characters` - 创建角色
- `POST /api/characters/{id}/like?userId=1` - 点赞

**测试数据示例**:
```
名称: 唐三
原型人物: 小明
头像: 🔥
称号: 千手修罗
等级: 50
星级: 5
攻击: 850
防御: 600
速度: 750
技能名称: 昊天锤
技能描述: 释放强大的力量，造成毁灭性打击
背景故事: 唐门外门弟子，拥有双生武魂...
```

### 4. 情侣空间 (Couple.vue)

**路由**: `/couple`

**功能模块**:

#### 4.1 心情日记
- ✅ 记录今日心情（10种 Emoji 可选）
- ✅ 查看今日心情
- ✅ 查看伴侣今日心情
- ✅ 心情历史记录
- ✅ 设置是否对伴侣可见

**操作流程**:
1. 切换到"心情日记"标签
2. 如果今天没记录，点击"记录心情"
3. 选择心情 Emoji（😊😢😡😴🥳😍🤔😭😎🤗）
4. 填写今天发生的事
5. 点击"保存"
6. 左侧显示自己的心情，右侧显示伴侣心情
7. 下方显示历史记录

#### 4.2 相册墙
- ✅ 创建相册
- ✅ 查看相册列表
- ✅ 相册封面图标（Emoji）
- ✅ 情侣共享相册

**操作流程**:
1. 切换到"相册墙"标签
2. 点击"创建相册"
3. 填写相册标题、描述、封面图标
4. 点击"创建"
5. 相册卡片显示在网格中
6. 点击相册查看详情（功能开发中）

#### 4.3 留言板
- ✅ 发送留言给伴侣
- ✅ 查看留言记录
- ✅ 双向留言显示（左右对齐）
- ✅ 时间戳显示

**操作流程**:
1. 切换到"留言板"标签
2. 在输入框填写留言内容
3. 点击"发送"
4. 留言显示在列表中
5. 自己的留言在右侧（暖色背景）
6. 伴侣的留言在左侧（灰色背景）

**API 调用**:
- `GET /api/moods?userId=1&pageNum=1&pageSize=10` - 获取心情记录
- `POST /api/moods` - 创建心情记录
- `GET /api/moods/partner?userId=1` - 获取伴侣心情
- `GET /api/albums?userId=1` - 获取相册列表
- `POST /api/albums` - 创建相册
- `GET /api/messages?user1Id=1&user2Id=2` - 获取留言
- `POST /api/messages` - 发送留言

## 🎨 设计特色

### 暖色调主题
- 背景色: `#F7F4EF` (warm cream)
- 主色调: `#C4612F` (terracotta)
- 悬停色: `#A94E22` (darker terracotta)
- 淡色调: `#F2E3D6` (terracotta tint)
- 边框色: `#E7E1D7`

### 字体
- 标题: Font Serif（衬线字体）
- 正文: Inter（无衬线字体）
- 斜体强调关键词（terracotta 颜色）

### 交互设计
- 圆角按钮（`rounded-full`）
- 卡片悬停阴影效果
- 平滑过渡动画（transition）
- 弹窗遮罩（半透明黑色背景）
- 响应式网格布局

---

## 🔧 技术实现

### 状态管理
- 使用 Vue 3 Composition API（`ref`, `onMounted`）
- 固定测试用户 ID = 1（模拟登录）
- 伴侣 ID 自动计算（userId === 1 ? 2 : 1）

### API 对接
- Axios 封装在 `utils/request.js`
- 自动处理响应结构（`res.data`）
- 错误处理和提示
- baseURL: `http://localhost:8080/api`

### 表单处理
- v-model 双向绑定
- 表单验证（required）
- 日期选择器
- 数字输入（min 限制）
- 文本域（textarea）

### 分页功能
- 当前页码追踪
- 总页数计算
- 页码按钮切换
- 每页 10 条记录
