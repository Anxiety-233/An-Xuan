## 📝 快速启动

### 1. 准备数据库
```bash
mysql -u root -pAnxiety200695 < schema.sql
```

### 2. 启动后端
```bash
cd backend
mvn spring-boot:run
```

### 3. 启动前端
```bash
cd frontend
npm install
npm run dev
```

### 4. 访问应用
- 前端: http://localhost:5173
- 后端: http://localhost:8080
- 测试账号: `admin` / `admin123`
